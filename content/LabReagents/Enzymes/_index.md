{{% notice "info" %}}
Растворы ферментов готовятся только с добавлением mQ (автоклавированная ультрачистая вода типа I с сопротивлением 18МОм). Стоки ферментов рекомендуется фильтровать через 0.22 мкм фильтр и фасовать. Перед приготовлением стока необходимо обращаться к рекоммендациям производителя, чтобы подготовить оптимальный для данного белка раствор.
{{% /notice %}}
